CliniX is a smart health management system that connects patients,
doctors, nurses, and IoT devices into a single intelligent platform.

## Objectives
- Improve patient care and follow-up
- Provide instant access to medical records
- Enable real-time alerts for healthcare professionals
- Support AI-driven decision making

## Main Actors
- Patient
- Doctor
- Nurse

## Key Features
- Appointment booking and management
- Electronic medical records (EMR)
- Medication reminders and notifications
- Real-time health monitoring via IoT devices
- AI-based health risk prediction and alerts

## Technologies (Planned)
- Mobile application
- Backend REST API
- AI / Machine Learning module
- IoT integration
